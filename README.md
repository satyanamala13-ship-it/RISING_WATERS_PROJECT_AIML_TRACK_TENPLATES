# Rising Waters Project

## Project Description
Rising Waters is a Flood Prediction Web Application developed using Python and Flask. It predicts the possibility of flooding based on rainfall, humidity, and cloud cover values entered by the user.

## Features
- Predicts flood chances
- User-friendly web interface
- Built using Flask
- Simple and fast prediction

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
